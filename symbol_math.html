Module symbol_math

A symbolic math API.

This module is a simple library of tools to make symbolic computations and basic operations on algebraic functions.

Example

Example usage:

>>> import symbol_math
>>> f = symbol_math.Function("x^2+x+3x", "x")
>>> print(f.simplify())
x^2 + 4x
>>> print(f.derivative())
2x + 4
